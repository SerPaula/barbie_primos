Barbie Prime Checker ✨

¡Bienvenida al verificador de números primos más fabuloso de la web! Este proyecto combina matemáticas con la estética icónica de Barbie para ofrecer una experiencia interactiva, educativa y llena de estilo.

💖 Características

Diseño Barbie Core: Una interfaz vibrante con el clásico rosa Barbie (#ff69b4), utilizando una tipografía elegante y moderna con bordes redondeados y efectos de transparencia (Glassmorphism).

Lógica Matemática: Identifica instantáneamente si un número es primo utilizando un algoritmo optimizado basado en la raíz cuadrada.

Efectos Mágicos:

Si el número es primo: Una lluvia de estrellas brillantes y luminosas inundará la pantalla para celebrar el resultado. ✨🌟

Si el número no es primo: Aparecerán caritas tristes y nubes para indicar que el número no cumple con las condiciones de un número primo. 😢☁️

Totalmente Responsivo: Diseño adaptable que garantiza una experiencia perfecta tanto en computadoras de escritorio como en dispositivos móviles.

Interactividad: Soporte para teclado (tecla Enter) y animaciones fluidas para una mejor experiencia de usuario.

🚀 Tecnologías Utilizadas

HTML5: Para la estructura semántica del aplicativo.

Tailwind CSS: Para un estilizado moderno, rápido y responsivo sin archivos CSS externos.

JavaScript (ES6+): Para la lógica de verificación de números y el sistema dinámico de partículas animadas.

Google Fonts: Implementación de las fuentes Dancing Script (para el toque elegante) y Montserrat (para legibilidad).

📖 Cómo usar

Abre el archivo: Ejecuta el archivo index.html en tu navegador favorito.

Ingresa tu número: Escribe el número que deseas verificar en el campo de entrada central.

Verifica: Haz clic en el botón rosa "¡Verificar!" o simplemente presiona la tecla Enter.

Descubre la magia: Observa la animación resultante y lee el mensaje personalizado en pantalla.

👩‍💻 Instalación y Uso

Este es un proyecto de archivo único (Single File App). No requiere instalación de dependencias externas, servidores ni compiladores.

Copia el código fuente del archivo index.html.

Guárdalo en tu computadora.

Haz doble clic en el archivo para verlo funcionar.
